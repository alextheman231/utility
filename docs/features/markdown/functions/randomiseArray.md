# Function: randomiseArray()

> **randomiseArray**\<`ItemType`\>(`array`): `ItemType`[]

## Type Parameters

The type of the array items.

## Parameters

### array

`ItemType`[]

The array to randomise.

## Returns

`ItemType`[]
