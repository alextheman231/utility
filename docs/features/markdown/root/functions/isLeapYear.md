# Function: isLeapYear()

> **isLeapYear**(`year`): `boolean`

Checks if the provided year is a leap year.

## Parameters

### year

`number`

The year to check as a number.

## Returns

`boolean`

True if the year is a leap year, and false otherwise.

## Throws

If the year provided is not an integer.
