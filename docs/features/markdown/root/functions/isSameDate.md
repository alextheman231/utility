# Function: isSameDate()

> **isSameDate**(`firstDate`, `secondDate`): `boolean`

Checks if the provided dates happen on the exact same day, month, and year.

## Parameters

### firstDate

`Date`

The first date to compare.

### secondDate

`Date`

The second date to compare.

## Returns

`boolean`

True if the provided dates occur on exactly the same day, month, and year, and returns false otherwise.
