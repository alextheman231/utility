# Function: calculateMonthlyDifference()

> **calculateMonthlyDifference**(`firstDate`, `secondDate`): `number`

Calculates the monthly difference between two given dates, subtracting the first from
the second and gives a negative result if the first date occurs earlier than the second.

## Parameters

### firstDate

`Date`

The first date

### secondDate

`Date`

The second date

## Returns

`number`

The calculated difference as a number
