## Description of Changes

<!-- user-editable-section-start -->
- Add an `escapeHTML` function.
  - This escapes the following common HTML characters:
    - `&` -> `&amp;`
    - `<` -> `&lt;`
    - `>` -> `&gt;`
    - `"` -> `&quot;`
    - `'` -> `&#39;`

## Additional Notes

- This is expected to come in especially handy in `alex-c-line`, where HTML/Markdown templates are becoming an increasingly important part of the project.
- I think it's especially good to have for the dependency audits because they're meant to catch security issues with dependencies, and if a dependency is so insecure it contains HTML injection, we probably might want protection from that!
