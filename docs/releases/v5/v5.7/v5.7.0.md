This is a new minor release of the `@alextheman/utility` package. It introduces new features in a backwards-compatible way and should require very little refactoring, if any. Please read the description of changes below.
<!-- alex-c-line-release-summary-end -->

## Description of Changes

<!-- user-editable-section-start -->
- Deprecate the `RecordKey` type
    - TypeScript actually has its own type for this - `PropertyKey`
    - It is not worth us keeping and maintaining our own version of this anymore, as it is better to just use the actual source of truth for what the key type of `Record` is.

## Migration Notes

- Replace all occurrences of `RecordKey` imported from this package with the TypeScript-native `PropertyKey`.
- No need to import `PropertyKey` - it is built into TypeScript's global scope directly.
